# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus feedback on code and behavior, not people.

## Unacceptable Behavior

- Harassment, insults, or discrimination.
- Personal attacks or sustained hostility.
- Doxxing or publishing private information.

## Enforcement

Project maintainers may remove comments, reject contributions, or ban participants for violations.

## Scope

This code of conduct applies in project spaces (issues, PRs, discussions, and related channels).
